# AI Chatbot

This is an AI-powered chatbot application built with [Next.js](https://nextjs.org/), [pnpm](https://pnpm.io/), and other modern web technologies.

## Features
- Multi-screen chat interface
- Real-time communication with an AI backend
- Easy to set up and extend

## Prerequisites
Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher)
- [pnpm](https://pnpm.io/)

## Getting Started
Follow these steps to set up and run the project locally:

### 1. Install `pnpm` (if not already installed)
```bash
npm install -g pnpm
pnpm install
pnpm dev
